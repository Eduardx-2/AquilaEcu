# AQUILAECU
Es un proyecto desarrollado en Java y Python, cuyo objetivo es la extracción de información a partir de placas de automóviles. Los datos obtenidos provienen de fuentes abiertas y públicas (OSINT).
<p align="center">
  <img width="80%" height="145%" src="https://github.com/Eduardx-2/AquilaEcu/blob/main/nbproject/private/ecuador_data.png">
</p>

<h2>Caracteristicas</h2>

- Guarda la información en base de datos local.
- Conexión SQL con soporte a distintas bases de datos.
- Consulta masiva de placas.


## Instalación

```console
# La veesión más completa de este proyecto solo se encuentra disponible en las realeses del proyecto, descargue y compilelo usando java, la versión extensa fue desarrollada solo para linux.
# clonar el repo
$ git clone https://github.com/Eduardx-2/AquilaEcu.git

# Moverse al directorio
$  cd AquilaEcu

# Crea o inicia el entorno venv
$ [Linux] -> source /aquila/bin/activate
$ [Windows] python -m venv aquila -> .\aquila\bin\activate.exe


# Install
$ python -m pip install -r requirements.txt

# Ejecución
$ python main.py or python3 main.py

```
