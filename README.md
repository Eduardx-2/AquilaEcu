# AQUILAECU
Es un proyecto desarrollado en Java y Python, cuyo objetivo es la extracción de información a partir de placas de automóviles. Los datos obtenidos provienen de fuentes abiertas y públicas (OSINT).
<p align="center">
  <img width="80%" height="145%" src="https://github.com/Eduardx-2/AquilaEcu/blob/main/nbproject/private/ecuador_data.png">
</p>

<h2>Caracteristicas</h2>

- Guarda la información en base de datos local.
- Conexión SQL con soporte a distintas bases de datos.
- Consulta masiva de placas.
